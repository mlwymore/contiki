Tokens

The token system stores such non-volatile information as the manufacturing ID, channel number, transmit power, and various pieces of information that the application needs to be persistent between device power cycles. Tokens are stored in either internal EEPROM, external EEPROM, or Simulated EEPROM (in Flash).

Refer to micro/token.h for a detailed description of the token system.
Refer to token-stack.h for stack token definitions.
Refer to token-manufacturing.h for manufaturing token definitions.
