Contiki version: CVS 2009-11-23
Intended platforms: Tmote Sky
Tested platforms: Tmote Sky
Description:
A Tmote Sky application that measures the power consumption with
different radio duty cycles and sends the data to a gateway Tmote Sky,
which is connected to a PC. The PC runs a Java program (included in
this Contikiproject) that displays the power consumption data on
screen.
