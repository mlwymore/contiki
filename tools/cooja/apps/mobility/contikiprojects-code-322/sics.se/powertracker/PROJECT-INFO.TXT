Intended platforms: Tmote Sky in Cooja simulation
Tested platforms: Tmote Sky
Description:
A Cooja plugin that shows a window with the average duty cycle of the
Tmote Sky radio. This can be used to measure the power consumption of
the network as well as to assess the power overhead of a duty cycling
MAC protocol.
